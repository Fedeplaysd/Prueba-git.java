en este segundo texto voy a aclarar que antes de realizar la conexion a github debo primero hacer el git commit -m "primer commit", luego git branch -M main para que la rama sea "main" y ahi hago el git remote add origin https://github.com/Fedeplaysd/Prueba-git.java.git que es el link al repositorio de github, para finalizar con git push -u origin main, en donde origin es la conexion que se realizo a traves de git remote add origin https://github.com/Fedeplaysd/Prueba-git.java.git con el respositorio de github y el main es la correccion de nombre que se hizo a la rama con el comando git branch -M main.
